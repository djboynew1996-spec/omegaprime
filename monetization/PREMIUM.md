2. **ระบบสร้าง License Key** ด้วย HMAC-SHA256 ทันที

4. **เปิดใช้** ด้วยคำสั่ง:
   ```bash

   set OMEGA_LICENSE_KEY=OMEGA-<base64>-<sig>
   
   # หรือ save ลงไฟล์
   echo OMEGA-<base64>-<sig> > omega.lic

   dotnet run --project src/OmegaPrime.csproj
   ```
